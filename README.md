# Math

This is a personal project about plotting functions and math in general this is not for production use this is just for fun and learning purposes. atleast for now.

## Quick Start

1. Clone the repository
``` git clone https://github.com/JohnVictoryz/math.git ```

2. Install dependencies
``` pip install -r requirements.txt ```

3. Run the application
``` python src/main.py ```
